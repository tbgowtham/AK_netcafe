# AK Netcafe - Cyber Cafe Management System

A full-stack project featuring a React frontend, Node/Express backend, and SQLite database.

## Directory Structure
- `backend/`: Express server logic, routing, controllers, and SQLite database file.
- `frontend/`: React single page application with modern CSS dashboard.

## Setup and Running

1. **Install all dependencies:**
   ```bash
   npm run install-all
   ```

2. **Run in development mode (starts frontend & backend concurrently):**
   ```bash
   npm run dev
   ```

3. **Endpoints & Ports:**
   - Frontend: `http://localhost:5173`
   - Backend API: `http://localhost:5000`
